# GregTech: Brainrot Horizons

Ever thought that Gen Alpha and Gen Beta won't play GTNH because so many have no attention spans? No? Whatever, I did.

This silly mod adds a brainrot language to GTNH. Do note that not all names have been modified, I'm adding brainrot as I go. If you have any ideas open a PR modifying the Brainrottify class. This class automatically translates the original translation files from English into Brainrot. Conveniently I have also included a tool to export all the game's translations into export.lang, you can enable this in the mod config.

This mod is client side only, don't put it on servers.

### Note on translation the GregTech.lang files

GT translates some items through its own language handler, it allows adding languages by placing GregTech_lang_CODE.lang in the modpack directory. This mod provides a language file for that on first start. The vanilla language handler is also provided a lang file by this mod. When running brainrottify you should provide both the mod's export.lang and GregTech.lang in the "langfiles".

### Why some things aren't translated (yet)

Some things just aren't translated by gt_BR.lang and GregTech_gt_BR.lang, I'm not certain why yet and I don't want to spend more time on this than its worth. As an example Tungstensteel Ingot is brainrotted correctly but Tungsten Ingot is not.
